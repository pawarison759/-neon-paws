# Neon Paws: Pro Edition 🐾✨

Premium Arcade-style game with Neon Cyberpunk aesthetics. Catch the mice, use power-ups, and climb the Hall of Fame!

## Features
- **PWA Ready**: Install on mobile and play offline.
- **Login System**: Save your highscores and stats locally.
- **Customization**: 7 Neon colors for your cat.
- **Power-ups**: Speed Boost, Magnet, and Time Freeze.
- **Mobile Optimized**: Virtual joystick for smooth control.

## How to play
1. Open `index.html` in any modern browser.
2. Register or Login.
3. Use **WASD** or **Arrow Keys** (PC) or the **Joystick** (Mobile) to move.
4. Catch mice to score and build combos!

## Installation
Add to Home Screen on your mobile device for the full app experience.
